# Orlena – AI System to Increase Restaurant Revenue

## What this does
Orlena helps restaurants increase average order value (AOV) by understanding customer behavior, preferences, and dietary patterns.

Instead of just taking orders, Orlena intelligently nudges customers toward better combinations, upsells, and personalized recommendations.

## Why this matters
Most ordering systems are passive.

Orlena is proactive:
- Understands repeat customer behavior
- Learns preferences over time
- Accounts for dietary restrictions
- Suggests optimized orders that increase ticket size

## Core Capabilities
- Behavioral pattern recognition
- Personalized upselling
- Dietary-aware recommendations
- Customer preference tracking
- Order optimization engine

## Example
Customer: Orders a veg burger regularly  
Orlena: Suggests combo with fries + drink based on past behavior  

Customer: Lactose intolerant  
Orlena: Avoids cheese-based suggestions automatically
